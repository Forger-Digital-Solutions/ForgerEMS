Portable Apps (Windows / WinPE)

Run without installing
  • CrystalDiskInfo — SSD/HDD health, SMART, temps
  • HWiNFO — full hardware inventory + sensors
  • CPU-Z / GPU-Z — quick CPU/GPU/RAM reads
  • OCCT — stress testing (CPU/GPU/PSU/RAM)
  • DiskGenius — partitions + recovery (use carefully)
  • Macrium — imaging/cloning (if you keep a portable build)
  • DDU — GPU driver cleanup (safe mode recommended)
  • Rufus — create boot media quickly
  • AnyDesk / RustDesk — remote support

Offline Drivers (Fast Fixes)

Device Manager → Browse
  • Chipset — run first when needed
  • LAN/Wi-Fi — get online fast
  • NVMe/Storage — storage controller issues
  • Touchpad — restore gestures (ELAN/Synaptics)
