Full Setup Guide (No Script Required)

Click links, download tools, and place files into the correct folders. This page exists so the Visual README stays clean.

1) Install Ventoy (Do This First)

Ventoy must be installed to the USB before booting ISOs. Installing Ventoy may format the USB (one-time). Always back up first.

Warning: Double-check the drive letter in Ventoy2Disk before clicking Install/Update. Selecting the wrong drive can wipe the wrong disk.
⬇️
Ventoy Download (Official)
Download Ventoy, run Ventoy2Disk, select the USB, install.
📖
Ventoy Getting Started
Official docs: install steps, UEFI/BIOS notes, Secure Boot info.

2) Where Everything Goes

ISO\Linux\ → Ubuntu / Mint / Kali / SystemRescue (ISO files) ISO\Tools\ → Clonezilla / GParted / Rescuezilla / MemTest / Hiren’s / UBCD (ISO files) ISO\Windows\ → Windows 10/11 ISOs + WinPE ISO (Sergei Strelec) Drivers\ → Offline driver packs (folders with INF/CAT/SYS or installers) Tools\Portable\ → Portable Windows apps (no install) Tools\Portable\Disk\ → Disk health + recovery tools Tools\Portable\Hardware\ → CPU/GPU/RAM + sensors + stress tools Tools\Portable\Network\ → Network scanning / quick checks Tools\Portable\System\ → Sysinternals / BSOD / driver tools Tools\Portable\Remote\ → AnyDesk / RustDesk Tools\Portable\USB\ → Rufus / Ventoy / Etcher Tools\Portable\GPU\ → DDU / NVCleanInstall ForgerTools\ → Your Forger apps (DisplayForger, HardwareForger, etc.) MediCat.USB\ → MediCat folder toolkit (copied manually as a folder) _downloads\ → Updater staging folder _archive\ → Older versions kept by updater _logs\ → Update logs _reports\ → Diagnostic reports you generate

Tip ISO files can be named however you want, but clear names help (ex: Win11_23H2_x64.iso).

4) Drivers (Offline Fixes)

Store driver packs in Drivers\. When Windows has no network:

  • Open Device Manager
  • Right-click the missing device → Update driver
  • Browse my computer → select USB:\Drivers\

Order (when needed): Chipset → LAN/Wi-Fi → NVMe/Storage → Touchpad → rest.

5) Updates (Optional)

If your USB includes Update-ForgerEMS.ps1 + ForgerEMS.updates.json, you can update selected items automatically:

Set-ExecutionPolicy -Scope Process -ExecutionPolicy Bypass D: cd D:\ .\Update-ForgerEMS.ps1 -DriveLetter D

The updater downloads to _downloads, logs to _logs, and archives replaced files to _archive. Some downloads are shortcut links only (Windows ISOs, MediCat, etc.) because direct links change.
