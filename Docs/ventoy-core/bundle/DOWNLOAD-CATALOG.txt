ForgerEMS Download Catalog
==========================

The Ventoy core uses explicit catalog roles so the
manifest, folder contents, and UX all describe the
same business rule:

- active managed autodownload
- disabled but eligible for re-promotion
- info shortcut
- placeholder-only
- manual only
- review-first

Note: managed still depends on upstream availability.
It means the manifest points at an official direct
artifact with checksum coverage today. It does not
guarantee that the upstream will stay reachable or
unchanged forever.
