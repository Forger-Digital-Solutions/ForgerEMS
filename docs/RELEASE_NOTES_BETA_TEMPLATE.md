# Beta release notes — paste at top

**Recommended download (most testers):**  
`ForgerEMS-v1.1.11-beta.1.zip`

**Instructions**

1. Download the ZIP from the GitHub Release.  
2. Extract to a short folder path.  
3. Run **`START_HERE.bat`**.

**Advanced / direct installer:** the standalone installer EXE may trigger **stronger SmartScreen** prompts until the build is signed and gains reputation. Prefer the ZIP flow for first-time testers.

**Inside the ZIP (typical):**

- `START_HERE.bat`
- Installer executable
- `VERIFY.txt` (when shipped)
- `CHECKSUMS.sha256`
- `DOWNLOAD_BETA.txt` (when generated by release script)
- `release.json` (when shipped)
